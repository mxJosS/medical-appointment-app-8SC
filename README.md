# Medical Appointment App (8SC)

## Objetivo general
Desarrollar una aplicación web para la gestión de citas médicas que permita a los usuarios registrar, consultar y administrar citas, optimizando la organización y reduciendo tiempos de atención mediante un sistema centralizado y fácil de usar.

## Tecnologías a utilizar
- **GitHub Projects (Kanban)**: organización y seguimiento de tareas
- **Git / GitHub**: control de versiones y colaboración
- **Frontend**: Tailwind CSS, Flowbite
- **Backend**: Laravel
- **Base de datos**: MySQL
